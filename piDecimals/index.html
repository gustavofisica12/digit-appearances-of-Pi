Digit appearances of Pi

How this works?

This web application can analyze the first 15000 decimals of Pi and count how many times each digit (0 to 9) appear. Write in the box how many decimals (from 1 to 15000) you want to analyze.

Why this?

As everybody know, Pi has infinite decimals, starting with

This special number has been studied by a lot of people that tried to find patterns in its decimals, without success. The digits seems to appear randomly and symmetrically. This program was created to prove that this is indeed the case.


As you use the program, you will be able to realize that by increasing the number of decimals to analyze, all the digits appear symmetrically; none of them predominates or lags behind. It seems that all the digits are generated randomly and uniformly distributed!!! Make your own conclusions.
